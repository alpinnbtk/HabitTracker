## 📌 Deskripsi Proyek

Habit Tracker adalah aplikasi berbasis Android yang dirancang untuk membantu pengguna membangun dan mempertahankan kebiasaan baik (*good habits*) secara konsisten.

Aplikasi ini memungkinkan pengguna untuk mencatat aktivitas harian, memantau progres, serta melihat status pencapaian dari setiap kebiasaan yang dilakukan.

---

## 🎯 Tujuan Aplikasi

* Membantu pengguna membangun kebiasaan positif
* Memantau perkembangan habit secara visual
* Meningkatkan konsistensi pengguna dalam menjalankan aktivitas

---

## 🔐 Fitur Login

Pengguna dapat melakukan login untuk mengakses fitur dalam aplikasi.

---

## 🏠 Dashboard

* Menampilkan daftar habit yang dimiliki pengguna
* Setiap habit ditampilkan dalam bentuk card
* Informasi yang ditampilkan:

  * Nama habit
  * Deskripsi
  * Progress
  * Status (In Progress / Completed)

---

## ➕ Manajemen Habit

Pengguna dapat:

* Menambahkan habit baru
* Menentukan target (goal) habit
* Memilih icon untuk habit

---

## 📈 Progress Tracking

* Pengguna dapat memperbarui progress habit
* Progress ditampilkan dalam bentuk progress bar
* Status habit akan berubah menjadi **Completed** jika target tercapai

---

## 🏗️ Arsitektur

Aplikasi ini menggunakan arsitektur:

**MVVM (Model - View - ViewModel)**

---

## 🧭 Navigation

Menggunakan Navigation Component untuk mengatur perpindahan antar halaman dalam aplikasi.
